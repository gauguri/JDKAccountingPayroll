MVP 2 — Payroll

Adds payroll for both businesses: set up employees, run a pay cycle, print pay stubs, and produce payroll-tax reports for the CPA. Built on the same principles as MVP 1 — explain before you post, and tax rates live in versioned data, never hardcoded.

base: main compare: mvp2-payroll
⚠️ Sample tax tables. 2026 federal & NJ rates are seeded in draft status and flagged "review before use." They must be approved by a CPA before running a real payroll. JDK Books prepares and calculates — it does not file.

What's included

  • Versioned tax-rule engine — effective-dated tax_ruleset / tax_rule tables with a draft → approved workflow. No rate is hardcoded; every payroll records which ruleset versions it used. Includes an admin view and an approve endpoint.
  • Employees & employer setup — per company; SSN and direct-deposit details encrypted and masked everywhere (SSNs never enter the audit log).
  • Run-payroll wizard — period → hours → calculate → review breakdown → approve → post. Computes gross, federal & NJ withholding (bracket method), Social Security + Medicare (+ additional Medicare), and employer FUTA/SUTA, all with year-to-date wage-base caps.
  • Balanced journal entry on post — wages + employer taxes debited; net pay and payroll-tax liabilities credited. Books stay tied out (trial balance + balance sheet still balance).
  • Pay stubs & reports — per-employee stub PDFs, payroll summary, employee earnings, employer taxes, payroll-tax liability, and Form 941 / 940 worksheet data (data only).
  • Frontend — Payroll screen with Employees / Run payroll / Reports tabs and a prominent sample-rate warning.

How to test

# Full stack
docker compose up --build      # web :3000, api :8000/docs

# Backend tests
cd backend && pip install -r requirements.txt && pytest

test_payroll.py covers: sample rules seeded as draft, ruleset approval, SSN masking, run calculation (SS = 6.2%, Medicare = 1.45%, net = gross − taxes), books staying balanced after posting, pay-stub PDF generation, and the tax-liability report. Verified independently: a $20 × 80h biweekly check yields SS $99.20, Medicare $23.20, correct bracket withholding, a balancing journal entry, and correct wage-base capping.

Design guardrails honored

Tax rates as versioned data (approve-before-use) · explain/review before posting · books corrected via entries, never edited · "organize and calculate, don't file."

Out of scope (by design)

Automatic direct-deposit ACH (the spec defers this) and year-end W-2/W-3 forms. Next up: MVP 3 — sales tax, tax-document vault, and business/personal tax worksheets.
